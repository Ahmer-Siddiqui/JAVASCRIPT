Edon is a simple and elegant Bootstrap 5 Template. This template has 1 unique homepage layouts and responsive, easy to customize. This Template suitable for any kind of fashion store.

Edon comes with a variation of 21+ HTML pages where there are 1 unique home versions. It is built with Bootstrap 5, HTML 5, SASS, CSS3 and W3C validated markup. This SEO-friendly template comes with numerous features like Smooth Animations, Google Fonts, Font Awesome, Ionicon and many more.

  • index.html is for Home page one

  • blog-detail.html is for Blog Detail
  • shop.html is for Shop Page
  • shop-leftsidebar.html is for Shop Page
  • shop-rightsidebar.html is for Shop Page
  • shop-list-leftsidebar.html is for Shop Page
  • shop-list-rightsidebar.html is for Shop Page
  • product-gallery-left.html is for Product Detail
  • product-gallery-right.html is for Product Detail

  • product-affiliate.html is for Product Detail

  • wishlist.html is for Wishlist page
  • checkout.html is for Checkout page
  • cart.html is for Cart page
  • compare.html is for Compare page
  • contact.html is for Contact page
  • faq.html is for FAQ page
  • 404.html is for Error 404 page
